Useful Vim Commands
===================

Git commands (fugitive)
-----------------------
- :Gedit [SHA]
- :Gedit %:h        (going back)
- :Gwrite
- :Gread
- :Gremove
- :Gmove
- :Gstatus
  - Stage/unstage file: `-`
  - Invoke commit     : `C`

- :diffput = :dp
- :diffget = :do
- :diffupdate
- [c
- ]c

- :Ggrep
- :Glog
- :Gdiff
